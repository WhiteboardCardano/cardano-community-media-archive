# Cardano Community Media Archive

The **Cardano Community Media Archive** is an open-source preservation project
dedicated to storing, documenting, and sharing all multimedia design assets
created for major Cardano ecosystem events.

This includes raw production files, design elements, motion graphics, stage
visuals, video source files, templates, and supporting documentation used for:

- **Vietnam Cardano Summit 2022**
- **Vietnam Cardano Summit 2023**
- **Vietnam Cardano Summit 2024**
- **Vietnam Cardano Constitutional Convention**
- **Vietnam Catalyst Working Group 2024**
- **Vietnam Cardano Meetup 2024**
- *(More events may be added over time)*

Our mission is to ensure that these creative works remain accessible,
educational, and reusable for the Cardano community â€” today and in the future.

## ğŸ¨ Whatâ€™s Included in the Archive?

Each event collection may contain:

- **High-resolution design production files:**  
  PSD, PSB, TIFF, Illustrator, Figma exports, vector packs.

- **Motion graphics assets:**  
  After Effects projects, animation source files, overlays, templates.

- **Stage visuals & screen assets:**  
  LED screen compositions, animated backgrounds, show packages.

- **Video source files:**  
  Editable video elements used for event intros, transitions, explainers.

- **Image packs:**  
  Event photography, atmosphere shots, social media frames.

- **Documentation:**  
  Naming conventions, file structure guides, production notes.

Because these assets can be extremely large (gigabytes to terabytes), files are
stored using reliable external archival platforms and indexed here.
